## Exploration

Work originated from an inbox idea proposing a ZX Spectrum aesthetic for
Portal's TUI, which currently has no visual identity. The shape picked
up: a single coherent retro visual pass — bold saturated rainbow
primaries on a black canvas, a block-character PORTAL logo with each
letter a different colour, rainbow gradient separator lines, a coloured
block cursor (`▌`) that cycles colours on navigation, spaced uppercase
8-bit-style headers, heavy/double ZX-style borders framing the TUI, a
Manic Miner-inspired status bar, a small rainbow block accent on modals,
and a loading interstitial with the logo and a rainbow-block progress
bar. The note flags the black-background assumption across terminal
themes (light/dark) as an open validation question, and an animated
cycling-colour border as a possible-but-likely-overkill option.

The user confirmed this is one coherent pass shipped as a unit rather
than several independently shippable concerns — settled as a feature.
The constituent pieces (logo/typography, borders/layout, status bar,
loading screen) all serve the one identity and do not split into
independent deliverables. No additional files or imports were shared.
